Quản trị nội dung Bảo tàng số 3D

Thêm / sửa / xoá nội dung (ảnh, video, mô tả, liên kết) cho từng bảng trưng bày trong phòng 3D. Mỗi bảng khớp với một material trong model — mã bảng dạng Image_Board_01, Image_Board_Left_03, Image_Board_Top_12… Danh sách bảng đã nạp sẵn đầy đủ ở dưới; bạn chỉ cần điền nội dung. Bảng để trống sẽ giữ ảnh có sẵn trong model và không bấm xem được.

1. Kết nối GitHub

Dán fine-grained token chỉ cấp quyền Contents: Read and write cho repo Rei-1407/Web-DPT. Token được lưu trong trình duyệt này (localStorage), không gửi đi đâu khác ngoài GitHub.

Cách tạo token (bấm để xem)
  1. Vào GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
  2. Repository access: chọn Only select repositories → Rei-1407/Web-DPT.
  3. Permissions → Repository permissions → Contents = Read and write.
  4. Generate → copy token (dạng github_pat_…) → dán vào ô trên.

3. Âm thanh phòng 3D

Nhạc nền phát khi vào phòng 3D và âm báo khi mở popup sản phẩm. Khi popup mở, nhạc nền tự nhỏ lại để người xem tập trung. Người xem có nút 🔊 ở góc trên để bật/tắt. Để trống ô đường dẫn nhạc nền → dùng tiếng nền êm dịu tổng hợp sẵn; để trống ô âm báo popup → dùng tiếng “pop” mặc định.
